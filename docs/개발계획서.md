# 펌프 용량 계산기 개발 계획서

## 📅 프로젝트 개요

**프로젝트명**: 하수도 펌프 용량 계산기  
**목표**: 압송관로 및 일반양정 시스템용 펌프 용량 계산 웹 애플리케이션 개발  
**기간**: 2026년 4월 ~ 2026년 5월  
**주요 기술**: Python, Streamlit, Pandas, NumPy

## 🎯 주요 기능

### 1. 계산 모드
- **압송관로 모드**: 압송관로 시스템 특화 계산
- **일반양정 모드**: 일반 양정 시스템 계산
- **혼합 모드**: 두 시스템의 혼합 계산

### 2. 입력 항목
- 유량 (m³/h, L/s, GPM)
- 양정 (m, ft)
- 파이프 직경 및 길이
- 마찰 계수
- 효율 계수
- 안전 계수

### 3. 출력 결과
- 필요한 펌프 용량 (kW, HP)
- 예상 전력 소비량
- 운영 비용 추정
- 최적 펌프 크기 권장

### 4. 내보내기 기능
- PDF 보고서 생성
- Excel 파일 내보내기
- CSV 데이터 저장
- 계산 이력 저장
